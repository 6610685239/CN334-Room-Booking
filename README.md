# Room Booking System

ระบบจองห้องประชุมและห้องเรียน ภาควิชาวิศวกรรมไฟฟ้าและคอมพิวเตอร์ พัฒนาด้วยกรอบการทำงาน Django และทำงานภายใต้สภาวะแวดล้อมของ Docker 

## ความสามารถของระบบ (Features)

* การยืนยันตัวตน (Authentication): เข้าสู่ระบบโดยใช้บัญชีของมหาวิทยาลัยผ่านการเชื่อมต่อกับ TU REST API
* ปฏิทินตารางห้องว่าง (Calendar View): แสดงตารางสถานะของห้องแบบรายเดือน รายสัปดาห์ และสามารถลากเลือกช่วงเวลาบนปฏิทินเพื่อเริ่มการจองได้
* ระบบการจอง (Booking Module): ผู้ใช้สามารถเลือกห้องและเวลาที่ต้องการ โดยระบบมีกลไกตรวจสอบการจองเวลาซ้อนทับ (Conflict Detection)
* ระบบจัดการสำหรับผู้ใช้ (User Dashboard): ผู้จองสามารถตรวจสอบประวัติการจองของตนเอง และทำการยกเลิกรายการที่ยังไม่ถึงกำหนดได้
* ระบบอนุมัติและจัดการ (Admin Dashboard): ผู้ดูแลระบบสามารถพิจารณาอนุมัติหรือปฏิเสธคำขอจองห้องได้จากส่วนกลาง
* ระบบการแจ้งเตือน (Email Notifications): ระบบจะทำการส่งอีเมลแจ้งเตือนผู้ดูแลระบบเมื่อมีคำขอจองใหม่ และแจ้งเตือนผู้จองเมื่อสถานะการจองถูกเปลี่ยนแปลง
* การจัดการข้อมูลห้อง (Room Management): ผู้ดูแลระบบสามารถเพิ่ม แก้ไข หรือปิดการใช้งานห้องประชุมและห้องเรียนได้โดยตรง
* รายงานและสถิติ (Report Module): ระบบสรุปข้อมูลชั่วโมงการใช้งานและอัตราการใช้งานของแต่ละห้อง พร้อมฟังก์ชันส่งออกข้อมูล (Export) เป็นไฟล์ CSV สำหรับนำไปประมวลผลต่อ

## สิ่งที่ต้องมีก่อนการติดตั้ง (Prerequisites)

* Docker
* Docker Compose
* Git
